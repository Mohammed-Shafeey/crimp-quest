# Crimp Quest

A pixel-RPG styled climbing finger-strength training tracker. Log repeaters, max hangs, and pinch tests, level up, complete weekly quests, and collect gear — all running client-side with no backend.

- **[index.html](index.html)** — self-contained app (React, ReactDOM, and Recharts bundled inline). Open it directly in a browser or serve it via GitHub Pages.
- **[crimp-quest.jsx](crimp-quest.jsx)** — readable JSX source for the same app.

Data is stored locally in the browser; nothing is sent to a server.
